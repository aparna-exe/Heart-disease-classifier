# Heart Disease Diagnostic System 🩺
A Python-based diagnostic tool using **Gaussian Naïve Bayes** to predict heart disease probability.

## Features
- **Bayesian Inference:** Calculates posterior probability based on Age, BP, and Cholesterol.
- **GUI Interface:** Built with Tkinter for real-time user input.
- **Performance Metrics:** Includes a built-in Confusion Matrix and accuracy reporting.

## Performance
The model achieves high accuracy on curated clinical data, with **Zero False Negatives**—ensuring maximum patient safety during testing.
